# Ultimate Healer Assistant



## Goals

- Aid in the evaluation of healers.
  - Output
  - Efficiency
  - Triage 
  - Timing
- Provide insight on healer strengths and weaknesses for the raid planning phase.
- Help evaluate raid tenacity.
  - Defensive use and value
  - Avoidable damage
  - One-shots

## Ideals

- Program should be able to accomplish all goals without the use of an external server.
This will keep overhead minimal especially when using GitHub for distribution. 
- Additional quality of life features that require an external server should have an implementation that may work without one.
This may be technical and will not be guaranteed to function. 
- Connections to the internet should only be for functionality and will not include personal, telemetry, or other data.
- Updates should be as much as possible optional and tier specific information provided as a download pack, or external server API request.

## Monetization

- No ads.
- Strictly optional.
Program should accomplish all goals to the greatest ability possible without monetization.
- Features restricted behind monetization should only be ones that require additional overhead such as an external server.
- Ask for a donation once, and get out.
If users want to donate later they can find information in settings.
